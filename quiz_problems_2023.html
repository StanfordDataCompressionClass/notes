Quiz 11 solutions were not covered in lecture, so here they are!

1.1

Solution: For a one-bit quantizer, we need to divide the interval [0,1] into two parts and assign a reconstruction point to each. By symmetry, we can argue that the two intervals must be [0,0.5] and [0.5,1]. Now for the reconstruction points, we need to find a point that minimizes the ∫00.5​(x−x^)2dx for the first interval which is simply the mean value 0.25 (can verify that computing the integral and then finding the minima - in general for a different distortion we are looking for the "median" corresponding to that distortion, for MSE the median is the mean!). Similarly for the second interval the reconstruction point will be 0.75.

X<1/2, X>1/2 Reconstruction points = 0.25,0.75

1.2

Solution: Let us first compute the integral ∫0Δ​(x−2Δ​)2dx which evaluates to 12Δ3​. Now for our case the distortion will be given by the squared error across the range of x from [0,1] (note that the pdf is simply equal to 1 in this whole range). We can split into two intervals according to the reconstruction point and get ∫00.5​(x−0.25)2dx+∫0.51​(x−0.75)2dx Reusing the result we just computed with Δ=0.5, we get the answer as 2×120.53​≈0.0208

Solution: Here we are trading off between assiging two intervals to each dimension vs. providing all four intervals to a single dimension. We can reuse the result from above for this: ∫0Δ​(x−2Δ​)2dx=12Δ3​

Q1: Reverse water filling (3 points)

Note: The problem appears long, but it's mostly just a recap from class!

Recall the problem of compressing two independent Gaussian sources X1​,X2​ with means 0 and variances σ12​ and σ22​. For the squared error distortion we saw in class, the rate distortion function is given by

min21​(D1​+D2​)≤D​ 21​[(21​logD1​σ12​​)+​+(21​logD2​σ22​​)+​]

where (x)+​=max{0,x}.

The solution suggests we split the overall distortion between the two components and then use the optimal strategy for each component independently.

We also saw in class that the optimal split into D1​ and D2​ is given by a reverse water filling idea, which is expressed in equation as follows:

For a given parameter θ, a point on the optimal rate distortion curve is achieved by setting

Di​=min{θ,σi2​} for i=1,2 D=21​(D1​+D2​)

And the rate given by 21​[(21​logD1​σ12​​)+​+(21​logD2​σ22​​)+​]

This can be expressed in figures as follows (assuming without loss of generality that σ12​<σ22​):

When D is smaller than both σ12​ and σ22​, we choose both D1​ and D2​ to be equal to D.

When D exceeds σ12​ but is below 21​(σ12​+σ22​), we set D1​ to be σ12​, and choose D2​ such that the average distortion is D.

When D is equal to 21​(σ12​+σ22​) we can just set D1​=σ12​ and D2​=σ22​.

Now consider a setting with σ12​=1 and σ22​=3.

Q3: Predictive Coding (2 points)

You find a source where consecutive values are very close, so you decide to do predictive lossy compression. Encoder works in following fashion: it first transmits the first symbol and after that it quantizes the error based on prediction from last encoded symbol. The quantized prediction error is transmitted.

Formally, suppose X1​,X2​,… is your original sequence and X^1​,X^2​,… is the reconstruction sequence. Then we have:

  • for the first symbol the reconstruction X^1​=X1​, i.e., you are losslessly encoding the first symbol
  • prediction for Xn​ is simply X^n−1​
  • prediction error is en​=Xn​−X^n−1​
  • quantized prediction error is e^n​
  • reconstruction for Xn​ is X^n​=X^n−1​+e^n​
  • the transmitted sequence is X1​,e^2​,e^3​,…

For this question, assume that the quantization for the prediction error is simply integer floor.
